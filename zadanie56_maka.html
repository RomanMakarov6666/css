1. Składnia teoretyczna definiowania klasy

selektor.nazwa_klasy { cecha: wartość; }

2. Opis dla "cecha" oraz "wartość"

„Cecha” to atrybut CSS (np. color, font-size), a „wartość” to przypisanie tej cesze (np. red, 12pt).

3. Jaki znacznik może być selektorem? Trzy przykłady:

Znaczniki mogące być selektorami to:
p – akapit
h1 – nagłówek
td – komórka tabeli

4. Uwagi na temat nazwy klas

Nazwą może być dowolny pojedynczy wyraz, nie może on zawierać:
a) spacji
b) kropki
c) dwukropka
d) nawiasów
e) plusa
f) znaku równości
g) przecinka
h) pytajnika

Może zawierać:
a) litery (A-Z, a-z)
b) cyfry (0-9)
c) myślniki (-)
d) podkreślniki (_)

Nazwa nie może zaczynać się od cyfry ani myślnika.
Gdy chcemy użyć zakazanych znaków, należy poprzedzić je ukośnikiem odwrotnym (\).

5. Zastosowanie klas

Klas używamy do wspólnego stylowania różnych elementów bez względu na typ znacznika.

6. Zastosowanie listy klas

Stosuje się, gdy jeden element ma przyjąć kilka klas – class="klasa1 klasa2".

7. Składnia listy klas i uwagi

<selektor class="klasa1 klasa2">...</selektor>

Kolejność klas nie ma znaczenia, decyduje kaskadowość.

8. Podzbiory klas

selektor.klasa1.klasa2 { cecha: wartość; }

Stosujemy, gdy reguła ma dotyczyć tylko elementów mających wiele konkretnych klas jednocześnie.

9. Sygnał, że należy zastosować klasy

Gdy element występuje na wielu stronach i ma to samo formatowanie, warto użyć klasy.

10. Planowanie klas

Klasy planujemy już na etapie projektowania, by łatwo zarządzać stylem całej strony.

11. Cel komentarzy przy klasach

Pomagają pamiętać, do czego dana klasa była przypisana, szczególnie przy dużych arkuszach CSS.

12. Przykład definiowania klasy i użycia

CSS:
p.przyklad { color: red; }

HTML:
<p class="przyklad">To jest akapit.</p>

13. Klasa dla jednego selektora

CSS:
p.moja { color: green; font-size: 14pt; }

HTML:
<p class="moja">Treść akapitu</p>

14. Klasa uniwersalna i znak „*”

Znak „*” oznacza selektor uniwersalny. Można go pominąć.
Przykład: 

CSS:
.przyklad_uni { font-size: 12pt; color: red; }

HTML:
<p class="przyklad_uni">Przykład akapitu</p>
<h1 class="przyklad_uni">Przykład nagłówka</h1>
