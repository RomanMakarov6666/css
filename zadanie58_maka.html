1) Co to są pseudoklasy?

Pseudoklasy to specjalne klasy CSS, które umożliwiają stosowanie stylów do określonych stanów elementów HTML, takich jak odwiedzony odnośnik, pierwszy element dziecka, aktywny link itp. Dzięki nim możemy uzyskać dynamiczne efekty bez użycia JavaScriptu.

2) Uwaga dotycząca dwukropka

W pseudoklasach bardzo ważny jest dwukropek (:) umieszczony pomiędzy selektorem a nazwą klasy. Bez tego znakowania pseudoklasa nie zadziała poprawnie. Przykład: a:hover

3) Tabela opisująca pseudoklasy

Pseudoklasa Opis
a:link Styl dla nieodwiedzonego odnośnika.
a:visited Styl dla odwiedzonego odnośnika.
a:hover Styl podczas najechania kursorem na element (np. odnośnik lub div).
a:active Styl aktywnego elementu w czasie kliknięcia.
a:focus Styl aktywowanego elementu (np. poprzez klawisz tabulacji).
:lang Stylowanie dokumentów wielojęzykowych.
:first-child Styl pierwszego dziecka w elemencie nadrzędnym.

4) Zastosowanie pseudoklasy :active

Pseudoklasa :active pozwala stylizować elementy, gdy są kliknięte, ale zanim przycisk myszy zostanie puszczony. Może być używana przy:

  • odnośnikach, by nadać styl klikniętemu linkowi,
  • formularzach – poprawiając ich czytelność i interakcję,
  • przyciskach – wizualny efekt naciśnięcia.

5) Przykłady (trzy) kodów definiowania pseudoklas

Temat: Kolor odnośnika przed i po odwiedzeniu

a:link {
  color: red;
}
a:visited {
  color: green;
}

Temat: Stylowanie odnośników z efektem hover

a {
  font-size: 12pt;
  text-decoration: none;
}
a:link {
  color: navy;
}
a:visited {
  color: green;
}
a:hover {
  color: red;
  text-decoration: underline;
}

Temat: Stylowanie aktywnego odsyłacza przy użyciu :focus

a:focus {
  color: brown;
}
